Features

Everything you need for collaboration

Built for developers, designed for everyone. Our widget integrates seamlessly into any tech stack.

Contextual Comments

Click anywhere on the DOM to pin a comment. We intelligently attach it to the underlying element, so it stays put even if the layout shifts.

Threaded Replies

Keep conversations organized with nested threads. Resolve discussions when they're done to keep the workspace clean.

Framework Agnostic

Works with React, Vue, Svelte, or vanilla JS. Web Component support ensures styles don't bleed into your app.

Flexible Storage

Comes with adapters for LocalStorage, Firebase, and Supabase. Or write your own `StorageAdapter` in 50 lines of code.

Keyboard First

Power users feel right at home. Press C to toggle comments, Esc to close.

Fully Customizable

Change colors, fonts, and positioning via simple config options. Your brand, your style.

Ready to integrate?
It takes less than 2 minutes.

Install via NPM or drop in a script tag. Configure your storage provider and you're good to go.

TypeScript ready
Zero dependencies
~50KB gzipped
App.tsx
import { useEffect } from 'react';
import { initCommentWidget } from '@commentwidget/core';

function App() {
  useEffect(() => {
    // Initialize the widget
    const widget = initCommentWidget({
      primaryColor: '#6366f1',
      storage: new LocalStorageAdapter('comments')
    });

    return () => widget.destroy();
  }, []);

  return <YourApp />;
}
